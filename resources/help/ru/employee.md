# hovera — инструкция для сотрудника конюшни

> Добро пожаловать. Эта инструкция предназначена для ролей **Тренер / Сотрудник / Менеджер / Только просмотр** в панели конюшни `/app`. Полное руководство владельца доступно по адресу `/app/help` с аккаунта владельца / админа.

---

## 1. Вход

1. Перейдите на `https://app.hovera.app/app/login`.
2. E-mail и пароль — приглашение пришло от конюшни на почту; при первом клике вы задаёте собственный пароль.
3. После входа вы попадёте на стартовую страницу, соответствующую вашей роли.

### Сброс пароля

`https://app.hovera.app/forgot-password` или кнопка «Не помню пароль» на экране входа. Ссылка действует **60 минут**.

### 2FA (опционально)

Меню пользователя (правый верхний угол) → **«Двухфакторная аутентификация»** → отсканируйте QR-код приложением TOTP (Google Authenticator / 1Password / Authy). Сохраните резервные коды.

---

## 2. Роли в конюшне

В зависимости от роли вы увидите разный объём меню и действий:

| Роль | Что видите | Что можете |
|---|---|---|
| **Менеджер** | Всё, кроме настроек конюшни и сотрудников | Управляете календарём, счетами, клиентами, лошадьми |
| **Тренер** | Календарь, свои бронирования, своих лошадей, клиенты | Редактируете свои бронирования, добавляете активности к лошадям, которых тренируете |
| **Сотрудник** | Карточка лошади (журнал активностей), календарь только-просмотр | Вписываете чистку, кормление, выгул |
| **Только просмотр** | Всё в режиме чтения | Ничего не меняете |

> Если вы **ковал / ветеринар**, у вас отдельная инструкция (роль `vet` → см. инструкцию специалиста).

---

## 3. Ежедневные действия — Тренер

### 3.1 Дневной календарь

**Путь:** `/app/calendar` (день / неделя, группировка по манежу / инструктору).

- Кликните запись → открываете подробности (участники, лошадь, статус).
- Пустой слот → кликаете, добавляете новое бронирование (если роль позволяет).

### 3.2 Список бронирований

**Путь:** `/app/calendar-entries` — полная таблица. Фильтры: тип, статус, «только мои», «предстоящие».

Статусы: *Ожидает* → *Подтверждённая* → *Завершённая* / *Отменённая* / *No-show*.

После урока отмечаете статус:
- **Завершённая** — присутствовал,
- **No-show** — клиент не явился без отмены,
- **Отменённая** — клиент отменил (с причиной).

### 3.3 Ваши лошади

**Путь:** `/app/horses`. По умолчанию фильтр «Мои» (те, которых вы тренируете).

Карточка лошади → вкладка **Активности**:
- ➕ **«Добавить активность»** → тип (чистка / кормление / левада / другое), заметка.

> Последние 7 дней активностей видны владельцу в портале клиента.

---

## 4. Ежедневные действия — Сотрудник

Ваш главный раздел: **Карточка лошади → Активности**.

Вы записываете повседневные действия вокруг лошадей:

- **Кормление** — например, «6:00 – сено + овёс»,
- **Чистка** — время + примечания,
- **Выгул** — какая левада, сколько часов,
- **Прочее** — например, «Слетела подкова, позвонить ковалу».

Запись сразу попадает в timeline лошади. Конюшня и владелец видят.

> Если заметили что-то тревожное (хромота, кашель) — добавьте запись **и** напишите сообщение через карточку лошади → вкладка **Сообщения**. Конюшня получит уведомление на почту.

---

## 5. Ежедневные действия — Менеджер

У менеджера самый широкий доступ, за исключением системных настроек:

- **Календарь + бронирования** — полное управление, перемещение чужих записей,
- **Клиенты** — добавление, редактирование, генерация magic-ссылок к порталу,
- **Лошади** — добавление, редактирование, документы, здоровье,
- **Счета** — выставление, отправка письмом, отметка как оплаченных,
- **Абонементы** — продажа, отмена, редактирование срока действия.

Полное описание каждого модуля есть в инструкции владельца (`/app/help` с аккаунта owner/admin).

## 6. Сообщения и уведомления

### 6.1 Сообщения в карточке лошади

У каждой лошади есть чат между конюшней и владельцем. Вы можете:
- читать историю,
- написать сообщение (например, «Сегодня выгул 3ч» — но это также можно через Активности),
- прикреплять файлы (PDF/JPG/PNG, макс. 10 МБ каждый, до 5 файлов).

### 6.2 Email-уведомления

По умолчанию вы получаете письмо при:
- новом бронировании у вас (Тренер),
- ответе клиента на ваше сообщение,
- уведомлении от конюшни («Завтра приезжает ковал в 14:00»).

В меню пользователя → **«Уведомления»** можно отключить отдельные категории.

## 7. Язык интерфейса

Меню пользователя (правый верхний угол) → **Polski / English / Deutsch / Français**. Предпочтение сохраняется per пользователь — работает и при следующем входе.

> Клиенты видят портал на языке, установленном конюшней (или собственном, если такая опция доступна). Ваше переключение не влияет на то, что видят они.

---

## 8. Безопасность

- **Пароль** — мин. 8 символов; не используйте такой же, как в других сервисах.
- **Сброс** — `/forgot-password` (ссылка на почту, TTL 60 мин).
- **2FA** — настоятельно рекомендуется для ролей Менеджер / Тренер.
- **Выход** — меню пользователя → «Выйти»; сессия сама истекает после 8ч бездействия.

> **Никогда не передавайте свой пароль коллегам.** У каждого свой аккаунт — это важно для аудита (кто внёс неправильный счёт, кто записал активность).

## 9. Частые проблемы

| Проблема | Что делать |
|---|---|
| Не помню пароль | `/forgot-password` → письмо со ссылкой |
| Не приходит письмо для сброса | Проверьте спам; сообщите владельцу конюшни |
| Не вижу бронирования | Проверьте фильтр «только мои» — возможно, снимите его |
| Не могу редактировать — «нет прав» | Ваша роль этого не позволяет — попросите админа изменить |
| Клиент жалуется, что не получил письмо | Проверьте адрес в карточке клиента; попробуйте «Отправить повторно» |

## 9a. Новые модули, которые вас касаются

- **План кормления лошади** — во вкладке «План кормления» карточки лошади указано точно, кто что и в какое время должен есть. Добавление/редактирование через Тренера/Менеджера; Сотрудник читает и выполняет. Клиент видит это также в портале (прозрачность диеты).
- **Склад кормов** — `/app/feed-inventory`. Ежедневные выдачи: нажмите **«+ Складское движение»** → тип «Выдача / расход» → количество (например, 12 кг) → подтвердите. Остаток автоматически уменьшается. Когда остаток упадёт ниже порога — в сайдбаре рядом со «Склад кормов» появляется badge с количеством позиций в алерте.
- **Карта веса лошади** — Тренер/Сотрудник вписывает ежемесячное измерение во вкладке **Вес лошади** (кг + опционально обхват груди). Столбец «Изменение» показывает, толстеет ли лошадь (🟢) / худеет (🟡) / стабильна (⚪).
- **Галерея фото лошади** — вкладка **Галерея** с миниатюрами; клиент видит грид в портале.
- **Дашборд «Сегодня»** (`/app`) — 4 плитки сверху (Бронирования сегодня / Свободные денники / Просроченные процедуры / Неоплаченные FV) + таблица активных бронирований. Быстрый обзор того, что у вас сегодня.

## 9b. Что вы видите в панели — ваша роль

Сайдбар фильтруется. Что вы видите, зависит от роли:

| | Тренер | Сотрудник | Менеджер | Только просмотр |
|---|:-:|:-:|:-:|:-:|
| Лошади · Уход и здоровье · План дня · Бронирования | ✓ | ✓ | ✓ | ✓ |
| Клиенты | ✓ | — | ✓ | ✓ |
| Денники · Тарифы · Регулярные · Инструкторы · Манежи | ✓ | — | ✓ | ✓ |
| Склад кормов | — | ✓ | ✓ | ✓ |
| Специалисты · Шаблоны процедур | — | — | ✓ | ✓ |
| Счета · Абонементы · Отчёты | — | — | ✓ | ✓ |
| Bulk invoicing | — | — | ✓ | — |
| Настройки · Сотрудники | — | — | — | — |

> Отсутствие раздела в сайдбаре не означает ошибку — вашей роли он не нужен. Если нужно что-то изменить в недоступном разделе, попросите админа/менеджера.
